# In Search of Better Harm Reduction Data

This project analyzes existing data available for harm reduction target variables and identifies weaknesses in the current offerings.
It proposes a long-term data collection approach to harm reduction outcomes in order to work toward training a predictive model which
can be leveraged to assess where program implementation is to the most benefit.

## Outcomes

While drug-related death rates are the best-tracked data, a large gap in the needs of a training model is the lack of consistent
year-over-year data collection for predictor variables. A long-term (e.g., ten year) data collection initiative linked to the
funding and expansion of legal harm reduction programs in three to ten target locations would provide the relevant data necessary
to train a functioning model.
